# E-commerce Sales Analysis

## Project Objective

This project analyzes sales data using Python and creates visualizations to identify business insights.

## Tools Used

- Python
- Pandas
- Matplotlib

## Features

- Data Cleaning
- Data Analysis
- Bar Chart
- Line Chart
- Pie Chart
- Business Insights

## Folder Structure

data/
visualizations/
report/
main.py

## How to Run

Install dependencies

pip install -r requirements.txt

Run

python main.py

## Output

- Sales by Product
- Daily Sales Trend
- Region-wise Sales Distribution
- Business Insights
