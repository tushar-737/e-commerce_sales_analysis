# E-commerce Sales Analysis Report

## Introduction

The objective of this project is to analyze sales data and identify sales patterns using Python.

## Dataset

Rows: 100

Columns:

- Date
- Product
- Quantity
- Price
- Customer_ID
- Region
- Total_Sales

## Data Cleaning

- Checked missing values
- Removed duplicates
- Converted Date column into datetime format

## Analysis Performed

- Total Revenue
- Average Revenue
- Product-wise Sales
- Region-wise Sales
- Daily Sales Trend

## Visualizations

1. Bar Chart
2. Line Chart
3. Pie Chart

## Insights

- Highest selling product identified.
- Highest revenue generating region identified.
- Daily sales trend analyzed.
- Overall revenue calculated.

## Conclusion

The project demonstrates a complete data analysis workflow including loading, cleaning, analyzing, visualizing, and reporting using Python.
